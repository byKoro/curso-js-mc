Enums / Enumerações — tipos que representam “categorias fixas” (por ex., GameMode, Direction, WeatherType

Classes / Instâncias — por ex., Player, Entity, Dimension, Block, World, etc. Por meio dessas classes você interage com o mundo.

Eventos — há “before” e “after” eventos para muitas ações no mundo (por ex., um jogador usar um item, bloco ser colocado, entidade morrer).

Tipos auxiliares / utilitários — por ex., localização, permutações de blocos (BlockPermutation), volumes (BlockVolume), etc

Variáveis globais — por exemplo, world para acessar dimensões/pessoas etc.
