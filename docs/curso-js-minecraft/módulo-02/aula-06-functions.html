Aula 06 - Functions

Conceito

Uma function é um bloco de código que realiza uma tarefa específica. Você pode reutilizar esse código várias vezes chamando a função pelo nome.

Exemplo

function dizerOla() {
  console.log("Olá, mundo!");
}

dizerOla(); // Chamando a função

Funções com parâmetros

  • Parâmetros são valores que você passa para a função usar.
  • Permite que a função faça tarefas diferentes dependendo do que você passar.
function cumprimentar(nome) {
  console.log("Olá, " + nome + "!");
}

cumprimentar("Yuri");
cumprimentar("Bicho");

Nem sempre a função vai só imprimir algo; às vezes ela calcula e devolve um resultado.

function soma(a, b) {
  return a + b;
}

let resultado = soma(5, 3);
console.log(resultado); // 8

Funções que Retornam Valores

let multiplicar = function(x, y) {
  return x * y;
};

console.log(multiplicar(4, 2)); // 8

Funções também podem ser anônimas (sem nome) ou escritas de forma curta com arrow functions.

Função Anônimas

Funções Anônimas e Arrow Functions

Arrow functions são mais curtas e modernas, muito usadas em JS.

Arrow Functions

let dividir = (x, y) => x / y;

console.log(dividir(10, 2)); // 5

Funções dentro de Funções

function dobro(n) {
  return n * 2;
}

function triplo(n) {
  return n * 3;
}

function calcularTudo(n) {
  console.log("Dobro: " + dobro(n));
  console.log("Triplo: " + triplo(n));
}

calcularTudo(5);

Você pode ter funções chamando outras funções.
