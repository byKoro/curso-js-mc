Aula 05 - Estrutura de Repetição

Pra que servem os loops?

  • repetir uma ação várias vezes
  • percorrer listas de dados
  • rodar cálculos em série
  • processar entradas do usuário

loop com contador

para percorrer propriedades de objetos

for (let i = 0; i < 5; i++) {
  console.log(i);
}

let pessoa = {
  nome: 'Yuri',
  idade: 20,
};
for (let chave in pessoa) {
  console.warn(chave, pessoa[chave]);
}

executa pelo menos uma vez

let i = 0;

do {
  console.log(i);
  i++;
} while (i < 5);

for

while

do...while

for...of

for...in

let i = 0;

while (i < 5) {
  console.log(i);
  i++;
}

repete enquanto a condição for verdadeira

para percorrer arrays

let numeros = [10, 20, 30];

for (let n of numeros) {
  console.log(n);
}

Desafio 06

Faça um script que mostre a tabuada de um valor x

let x = 2;

for (let count = 10; count >= 0; count--) {
  console.warn(`${x} x ${count} = ${x * count}`);
}

let x = 5;
let resultado = 1;
while (x > 1) {
  resultado = resultado * x;
  x--;
}
console.warn(resultado);
